# Terminal-tetris

一个运行在终端里的俄罗斯方块游戏，使用 Python 内置 `curses` 库实现，无需额外依赖。

## 支持

| Rotation System | 7-Bag | Hold | Ghost | Lock Delay | Block color | Marathon |
| --------------- | ----- | ---- | ----- | ---------- | ----------- | -------- |
| SRS             | Yes   | Yes  | Yes   | Yes        | Guideline   | Yes      |
